# Early development
Complete version of "System77".
"Hack"-and-slash platformer game with Unix-like terminal made in Godot Engine.

Still in the early development.

# Opening the project
1. Download `Godot Engine 4.5-dev2`
2. Download this repo `git clone https://github.com/RavioliMavioli/malware-slayer`
3. Import `project.godot` to `Godot Engine 4.5-dev2`
