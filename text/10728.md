## 6. Pull Request Process

To submit a contribution, fork the repository on GitHub. Make your changes in your forked repository, and make a feature branch. After completing all local modifications push the modified files up into GitHub

Then, create a pull request from your feature branch to the main repository, following our established branching rules:
```bash
git checkout -b feature/your-feature-name
# Make your changes
git commit -m "feat: Your meaningful commit message"
git push origin feature/your-feature-name
```
After creation of your pull request please allow ample time to be spent evaluating your code by the maintainers!
