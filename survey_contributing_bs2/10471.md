## **2. Getting Started**

Before contributing, it is essential setup a local development environment to test the project' functionalities.
	
To do this, clone the Awesome! repositories on GitHub using following command:
```bash
git  clone https://github.com/AwesomeOrg/AwesomeProject.git 
```
Next, navigate to the AwesomeProject root directories with this code :
```bash
cd AwesomeProject
```

Install required dependencies using the relevant tool:

```bash
npm install or pip install -r requirements.txt (if it has this requirement list!)
```

The default branches for contribution in the AwesomeProject project follows the "develop" branches. This provides separation development activities between main branch, intended as final, deployed, state, and its other contributors development efforts! Always pull from `develop`. 

Before working begin creating the new features you need, first, make sure it synchronised latest development code by performing an pull request
```bash
git pull origin develop
```
